**Software Environment**
- Python 3.13.7
- NumPy 2.4.1
- Implemented on MacOS Tahoe 26.5.1, but should run on other OS.


**Repository Structure**

In the repository the code structure is split into two primary parts:

- Root folder: Code, analysis and results of the altered baseline and coevolutionary setting.
    - Data output is found in the 'results' folder.
    - Data analysis is found in 'analysis.ipynb'.

- The 'OG-Yang\_baseline' folder containing the recreation of the Yang Baseline.
    - Data output is found in the 'yang\_reproduction\_outputs' folder.
    - Data analysis is found in 'analysis.ipynb', **inside the 'OG-Yang\_baseline' folder**.

**Run Commands**

To run the experiments for the altered baseline and coevolution see 'experiment\_runner.py' in the root folder. Next, to run the original recreated baseline see 'yang\_og\_baseline.py' in the 'OG-Yang\_baseline' folder.

**Additional Figures**

The figures that are to be reproduced can be found in the analysis notebooks. More figures than used in the report can be found here.
